# Quran Landing Page

A modern and responsive Quran Landing Page built with **HTML**, **CSS**, and **JavaScript**, enhanced with smooth animations and interactive user experience. The project focuses on clean design, accessibility, and engaging visual effects to create an attractive landing page.

## Features

- Responsive design for all screen sizes
- Smooth CSS animations and transitions
- Interactive JavaScript functionality
- Clean and modern UI
- Optimized layout and performance
- Easy-to-customize structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
